# Codex Prompt — Phase 0: CosyVoice2 + RHO-1 数据准备与分层

## 1. 项目背景

我正在做一个基于 **CosyVoice2 + RHO-1** 思路的实验工程，但你这次 **只需要实现 Phase 0**。

本项目的大方向是：
- 使用 `D_train` 作为正式训练集；
- 使用统计筛选得到的高质量子集 `D_ref` 作为 reference 数据；
- 后续在 CosyVoice2 的 **text-speech LM** 阶段做 selective training。

但是现在 **不要进入后续阶段**。  
当前任务仅限于 **Phase 0：数据准备与分层**。

## 2. 本次任务的唯一目标

实现一个 **最小可行、结构清晰、可复用** 的数据准备模块，完成以下事情：

1. 读取原始数据；
2. 明确区分并组织：
   - `D_train`：原始正式训练集
   - `D_ref`：统计筛选后的高质量子集
3. 对所有音频统一调用 **CosyVoice2 speech tokenizer**
4. 为每条样本产出统一表示：
   - 文本 token：`y`
   - speech token：`μ`
5. 将每条样本整理为统一结构：
   - `(y, μ)`
6. 产出可供后续 Phase 1 / Phase 2 / Phase 3 使用的标准化中间数据文件或索引文件

## 5. 你要先理解的核心事实

请基于以下事实实现：

1. `D_train` 是正式训练集，包含全部可用语音-文本对；
2. `D_ref` 是统计筛选出的高质量子集，来自 `D_train` 或同源语料；
3. 当前 Phase 0 的目的，是让 reference model 和 target model 将来比较的是 **同一套 speech token**；
4. 因此所有音频都必须经过 **同一个 CosyVoice2 speech tokenizer**；
5. 每条样本最终统一为 `(y, μ)`；
6. 这一步的本质是“统一 token 表示 + 数据分层 + 为后续阶段准备标准输入”。
